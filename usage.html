Spike Analysis

The 27–28 April spike accounts for 46% of total sessions and 66% of total cost. These were cron-driven orchestrator runs using the deepseek-v4-pro model for Planka Kanban review cycles ($1.74/M input, $3.48/M output). After switching to deepseek-v4-flash on 28 April ($0.14/M input, $0.28/M output), daily cost fell from $10 → ~$1/day — a 90% reduction.

Methodology

Costs are estimated from message count — each message assumed ~800 tokens (500 prompt + 300 completion). Pricing from the Hermes models cache: deepseek-v4-flash at $0.14/M input + $0.28/M output, deepseek-v4-pro at $1.74/M input + $3.48/M output. Actual API billing from DeepSeek's dashboard would give exact figures — the user has only ever topped up $40 total, so the $22 estimate is consistent with real spend. Discrepancies may come from prompt caching (discounts), free tier credits, or per-session token variance not captured by the fixed per-message estimate. Session data from local Hermes agent logs; automated cron jobs and gateway integrations are included.
